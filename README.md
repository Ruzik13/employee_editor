# employee_editor

Простое приложение для управления списком сотрудников: просмотр, добавление, редактирование и удаление записей.

## Как запустить
`npm  install` - устанавливаем зависимости

`npm run dev` - запускаем приложение

## Компоненты
Были реализованы следующие компоненты:
* CustomTable - компонента таблицы. Принимает массив заголков `heads` и массив данных `data`,  которые необходимо отобразить. Для взаимодействия с полями таблицы используется emit `select-bar`.  Пример использования:
  ```vue
  <CustomTable
    :heads="['Имя', 'Фамилия', 'Стаж', 'Возраст', 'Адрес']"
    :data="users"
    @select-bar="openEditForm"
  />
  ```
*  CustomModal - компонента модального окна. Разработан с помощью слотов. В качестве аргументов принимает `isOpen`, чтобы понимать, открыта модалка, или нет. При необходимостит добавления заголовка окна можно добавить template с именем `header`. Для контента используется template с именем `content`. Для закрытия при клике наружной части используется emit `close`.  Пример использования:
  ```vue
  <CustomModal>
    :isOpen='isModalOpen'
    @close="handleClose"
  >
    <template  #header>
      <h1>Здравствуйте!<h1>
    </template>
    <template #content>
      <p>Lorem ipsum</p>
    </template>
  </CustomModal>
  ```
* CustomInput - компонента поля ввода. В качестве аргументов принимает `placeholder` для внутренней подсказки, `label` для подписи инпута, `error` для вывода в случае ошибки. Пример использования:
  ```vue
  <CustomInput
          label="Адрес:"
          :error="errors.address"
          v-model="user.address"
        />
  ```
* CustomButton - компонента кнопки. В качестве аргумента принимает `variant` для вварианта отрисовки кнопки (default, outline, danger). Пример использования:
```vue
<CustomButton
      @click="openAddForm"
    >
      Добавить
    </CustomButton>
  ```

  Для хранения данных пользователей и функций для взаимодействия с ними используется хранилище `userStore`. Функции для взаимодействия:
  * `addUser` - для добавления пользователей
  * `editUser` - для редактирования пользователя
  * `deleteUser` - для удаления пользователя
